# 功能说明文档

## 系统概述

高校实验室设备库存与借用管理系统用于管理实验室设备、库存数量、借用申请、审核归还和操作日志。系统按管理员和普通用户两类角色展示不同功能菜单。

## 登录与权限控制

- 用户通过用户名和密码登录。
- 登录成功后将用户 ID、用户名、角色和真实姓名保存到 Session。
- 未登录访问业务页面会跳转到登录页。
- 普通用户不能访问 `/admin/**` 管理页面。
- 用户可以退出登录，退出后 Session 失效。

## 管理员功能

- 用户管理：新增用户、修改用户信息、删除用户、修改角色、停用用户、重置密码。
- 设备分类管理：新增分类、修改分类、删除分类、按分类名称查询。
- 设备信息管理：新增设备、修改设备、删除设备、按名称、编号、分类、状态筛选。
- 借用管理：查看全部借用记录，审核待审核申请，拒绝申请，办理设备归还。
- 操作日志：查看用户登录、设备维护、借用归还、数据导出等记录。
- 数据导出：导出设备信息 Excel 文件。

## 普通用户功能

- 设备查询：查看当前可借设备，按设备名称、编号、分类筛选。
- 借用申请：选择设备、借用数量、预计归还日期并提交申请。
- 我的借用：查看本人申请记录、审核状态、借出时间和归还时间。
- 修改密码：输入原密码和新密码完成密码修改。

## 借用业务规则

- 普通用户提交申请后，借用记录状态为“待审核”，此时不扣减库存。
- 管理员审核通过后，系统扣减设备可借数量，并将记录状态改为“已借出”。
- 如果库存不足，审核通过会失败。
- 管理员办理归还后，系统恢复可借数量，并将记录状态改为“已归还”。
- 维修中、停用、库存为 0 的设备不能被普通用户申请借用。
